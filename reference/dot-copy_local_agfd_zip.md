# Copies local AGFD file to tempdir() so the org file isn't touched

Copies local AGFD file to tempdir() so the org file isn't touched

## Usage

``` r
.copy_local_agfd_zip(x)
```

## Arguments

- x:

  A filepath pointing to the AGFD file provided by the user

## Value

A character vector of file paths for AGFD netCDF files
