# Create and apply a color data.frame for the clum_50m_2023_v2 data

Creates a [`data.frame()`](https://rdrr.io/r/base/data.frame.html) that
contains the hexadecimal colour codes that correspond with the integer
values to display the map colours as published by ABARES for the
Catchment Level Land Use (clum) data. Values are derived from the .qml
file provided by ABARES.

## Usage

``` r
.create_clum_50m_coltab()
```

## Examples

``` r
.apply_clum_50m_col_df()
```
